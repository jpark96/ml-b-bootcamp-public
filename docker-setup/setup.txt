=== DOCKER SETUP ===
Download Docker Toolbox at https://www.docker.com/products/docker-toolbox.

Follow instructions at:
Mac OSX - https://docs.docker.com/toolbox/toolbox_install_mac/
Windows - https://docs.docker.com/toolbox/toolbox_install_windows/

=== IPython Setup ===
0. Run Docker Quick Start Terminal to start your Docker terminal
1. Make sure your VM is running:
  docker version
2. If it says "Cannot connect to the Docker daemon. Is 'docker -d' running on this host?", run:
  docker-machine rm default
  docker-machine create default --driver virtualbox
3. Set up Docker container with needed resources (NumPy, SciPy, Pandas):
  docker -d -p 8888:8888 jupyter/datascience-notebook start-notebook.sh
4. Check your "DOCKER_HOST"  with "docker-machine env default"
5. Set your environment variables 
  eval $(docker-machine env default)
6. Go to your default browser and type "https://{VM_host_id}:8888"
7. Enjoy Jupyter Notebook!

=== Get Assignmetns ===
1. Open "New > Terminal" within your IPython browser
2. Clone the ml-b-bootcamp repository for assignments
  git clone https://github.com/jpark96/ml-b-bootcamp.git
3. Change directories to "ml-b-bootcamp/assignments"
4. Open the assignment!
